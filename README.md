# OnlineIncense
オンラインで芳名録の記入、香典の送付ができるアプリです。
App Storeには公開していません。


## ライブラリ
### CococaPods
・Firebase/Analytics
・Firebase/Auth
・Firebase/Firestore
・Firebase/Storage
・PKHUD
・RxSwift
・RxCocoa

### Swift Package Manager
・[Omise SDK](https://github.com/omise/omise-ios.git)
・[CurlDSL](https://github.com/zonble/CurlDSL)
